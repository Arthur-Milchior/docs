class jina.peapods.peas.remote.ssh.RemoteSSHPea(args)[source]¬∂

Bases: jina.peapods.peas.BasePea

Simple SSH based RemoteSSHPea for remote Pea management

Note

It requires one to upload host public key to the remote 1. ssh-keygen -b 4096 2. scp ~/.ssh/id_rsa.pub username@hostname:~/.ssh/authorized_keys

Note

As the terminal signal is sent via send_terminate_signal() from BasePea, there is no need to override/implement close() method. Lifecycle is handled by BasePea.
